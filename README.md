# Markdown Editor

A simple, focused markdown editor for macOS.

This application has been simplified from FSNotes to focus solely on markdown editing without the note management features.

## macOS app

### Key features

- **Markdown-first** editing experience
- **Fast and lightweight**
- **Syntax highlighting** within code blocks. Supports over 30 programming languages.
- **In-line image** support
- **Live preview** with toggle support
- **Dark mode**
- **Mermaid and MathJax** support

## Original FSNotes

This is a simplified version of FSNotes. For the full-featured notes manager with multi-folder storage, cloud sync, encryption, and more, see the original FSNotes project.
